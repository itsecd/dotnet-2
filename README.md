* В дирректории проекта запустить окно командной строки (cmd/powershell)
* Выполнить команду: ./ngrok config add-authtoken 28LQSAvb8JBlRtAlppbFCyTeSC9_7M71Hx6FD17PaR2CFBgZh
* Выполнить команду: ./ngrok http 443
